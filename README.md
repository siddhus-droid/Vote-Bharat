# 🗳️ Vote Bharat – Online Voting System

A secure web-based online voting system developed for conducting digital elections.

## 🚀 Live Demo
https://vote-bharat-style.vercel.app

## ✨ Features
- Secure voter authentication
- Admin dashboard
- Digital voting
- Vote counting and result display
- Responsive user interface

## 🛠️ Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS

## 📦 Installation

```bash
npm install
npm run dev
```
